# Contributing

Thanks for your interest!

## Getting Started
- Install deps with `pnpm install`
- Run `pnpm dev`

## Branching
- Use `feat/*`, `fix/*`, `chore/*` prefixes.

## Commit Messages
- Follow Conventional Commits.

## Testing
- Add tests for new features (if/when test setup is added).

## PRs
- Keep PRs focused and small.
- Link related issues.
